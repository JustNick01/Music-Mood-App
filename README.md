# Music-Mood-App

MusicMoodApp este o aplicație Java care redă muzică în funcție de starea de spirit a utilizatorului. Oferă o interfață modernă, suport pentru fișiere audio WAV/MP3, controale de redare și un sistem intuitiv de navigare între diferite mood-uri.

# Funcționalități

Selectarea unei stări (Mood): Happy, Sad, Chill, Energetic, Love, Focus\
Player audio integrat:\
    Play / Pause\
    Stop\
    Control volum\
Interfață grafică modernă realizată cu Java Swing + FlatLaf\
Încărcarea automată a melodiilor din folderul music/\
Animații și efecte vizuale în panouri\
Funcționează ca JAR sau EXE (cu JRE inclus)

# Cum se rulează aplicația


Rulezi MusicMoodApp.exe\
Nu necesită instalarea Java (aplicația conține propriul JRE)

# Adăugarea melodiilor

Mergi în folderul music/ \
Deschide categoria dorită (happy/, sad/, etc.)\
Copiază melodiile tale .mp3 sau .wav\
Aplicația le va detecta automat
